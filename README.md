# qLDPC

## Experimenting with QLDPC Codes

### Trivariate Bicycle or TB Codes
Based on (https://arxiv.org/abs/2507.09690), a formalism is shown to generate small qLDPC codes. The automated code for generation is given under "TB_QLDPC.py." So far, the code only provides the n and k parameters of the generated qLDPC code. The distance has not yet been done.
